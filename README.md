# Langton's Ant - Υλοποίηση και Πειραματισμοί

Ατομική εργασία για το Langton's Ant, με χρήση Python και της βιβλιοθήκης `turtle`.

## Περιγραφή

Μέσα στην εργασία παρουσιάζεται η βασική συμπεριφορά του Langton's Ant και στη συνέχεια εξετάζονται κάποιες παραλλαγές πάνω στον βασικό αλγόριθμο.

Σκοπός της εργασίας είναι να δούμε πώς επηρεάζεται η συμπεριφορά του μυρμηγκιού όταν αλλάζουμε την αρχική θέση, τον προσανατολισμό, το αρχικό πλέγμα ή προσθέτουμε ένα επιπλέον χρώμα.

## Περιεχόμενα εργασίας

Στην εργασία υπάρχει:

- Η βασική συμπεριφορά του Langton's Ant.
- Εισαγωγή δεύτερου μυρμηγκιού και πειραματισμός με την αρχική θέση και τον προσανατολισμό του.
- Μελέτη της περίπτωσης όπου το πεδίο από το οποίο ξεκινάει το μυρμήγκι έχει διαφορετικό χρώμα.
- Προσθήκη ενός τρίτου χρώματος.

## Αρχεία

Το repository περιλαμβάνει το τελικό PDF της εργασίας και τους κώδικες Python που χρησιμοποιήθηκαν για τις υλοποιήσεις και τους πειραματισμούς.
